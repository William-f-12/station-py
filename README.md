# Decentralized Instant Messaging Station (Python)

Demo project for [dimp](https://github.com/dimchat/core-objc) package.

## Usages

1.) Install Requirements

```
pip3 install dimp
```

2.) Run Server

```
cd station-py
python3 station/start.py 
```

3.) Run Test Client

```
cd station-py
python3 tests/client.py
